# 审计概览看板 (Audit Dashboard)

此看板是数字员工（AI Agent）运行状态与安全风险的核心监测中心。它为 IT 运维和安全团队提供全局视图，协助管理智能化应用的运行现状。

![审计概览](../pictures/审计概览.png)

---

## 🌟 核心功能与价值

在接入大量内部机器人或自动化任务后，审计仪表盘能够帮助您快速：
1. **掌握资源活跃度**：实时展示当前数字员工的并发请求及会话数量。
2. **洞察风险分布**：分类显示各种异常和高危行为的总体占比及时间线趋势。
3. **定位热点实体**：指出调用频率最高的用户以及高频触发安全防御的实体及工具。

---

## 📸 看板核心组件分析

根据系统的真实监控指标，面板被划分为上下多个维度的分析模块：

### 1. 多时间窗核心指标 (Key Metrics Overview)
以“今日 / 本周 / 本月”的时间分片清晰列示核心用量数据：
- **会话总数 / 活跃会话数**：涵盖全历史记录与设定窗口期内仍有请求的心跳数量。
- **用户访问数 / 设备连接数**：精准查验接入的独立账号 (AccountId 去重) 以及对应的接入终端 (Channel + Device 去重)。

### 2. 风险大盘与全库统计 (Risk Statistics & Live Data)
- **风险红绿灯**：统计并直接量化全库产生的高风险、中风险、低风险级别的违规或报错操作数。
- **风险大盘占比**：计算命中风险策略的污染会话在总会话池中的占比。
- **实时预警**：同步展示当前正在存活的连线状态，并抓取近 24h 内产生过“异常断开” (Aborted) 的会话。

### 3. 可视化趋势图表 (Visualizations)
将复杂的监控数据转化为直观图表：
- **风险操作结构图（饼图）**：对告警日志流按安全定级进行成分拆解，直观对比高风险与中低风险的比例结构。
- **近 7 日会话量趋势（折线图）**：监控系统的每日并发调用走向，判断是否出现系统级的峰值拥堵。
- **近 7 日风险操作趋势（平滑曲线图）**：根据时间轴跟踪中、高、低三种风险拦截动作的高峰期，分析系统暴露攻击的集中时间。

### 4. Top 排行榜 (Top Insights)
为安全治理提供“由上至下”的处置清单：
- **高频用户**：触发并发控制或建立对话最多的账号。
- **高频访问设备**：流量来源最大的通道。
- **高频命中风险**：识别引发错误拦截的根源（通过优先提取 `message_tool_name` 等工具属性），帮助锁定那些经常报错的下游接口或意图。
